# Telco Customer Churn Analysis
 
> Bir telekomünikasyon şirketinin müşteri kaybını
> tahmin eden uçtan uca veri bilimi projesi.
 
## Proje Hakkında
IBM Telco Customer Churn veri seti (7.043 müşteri) üzerinde
keşifsel veri analizi, özellik mühendisliği ve
makine öğrenmesi uygulandı.
 
## Kullanılan Teknolojiler
Python · Pandas · NumPy · Matplotlib · Seaborn · Scikit-learn
 
## Model Sonuçları
| Model | AUC |
|-------|-----|
| Logistic Regression | ~0.85 |
| Random Forest | ~0.84 |
| Gradient Boosting | ~0.86 |
 
## Temel Bulgular
- Aylık sözleşmeli müşteriler en yüksek churn riskini taşıyor
- İlk 12 aydaki müşterilerin yarısı ayrılıyor
- Online güvenlik hizmeti alanların churn oranı belirgin düşük
